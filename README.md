# Vplme Tap

## How do I install these formulae?

`brew install vplme/tap/<formula>`

Or `brew tap vplme/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "vplme/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
